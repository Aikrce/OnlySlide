# Frameworks

此目录用于存放项目使用的预编译框架（XCFrameworks）。

## 目录结构

- `ThirdParty/`: 存放第三方依赖的预编译框架
- `Custom/`: 存放自定义框架

## 使用说明

1. 所有框架都应该采用 XCFramework 格式
2. 每个框架都应该包含其版本信息
3. 更新框架时，保持向后兼容性

## 版本控制
- 框架更新时，请在此文件中记录更新信息
- 记录格式：框架名称 - 版本号 - 更新日期 - 更新说明

## 注意事项
1. 确保所有框架都经过完整测试
2. 保留每个框架的文档链接
3. 记录任何已知的限制或问题
