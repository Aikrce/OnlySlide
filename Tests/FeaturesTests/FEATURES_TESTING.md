# 功能模块测试指南

## 概述

功能模块测试专注于验证OnlySlide中各个独立功能模块的正确性。每个功能模块都应该有其对应的测试套件，确保模块的功能完整性和稳定性。

## 测试范围

- 幻灯片编辑功能
- 主题管理
- 导出功能
- 多媒体处理
- 协作功能
- 版本控制

## 测试结构

每个功能模块的测试应包含：

1. 基本功能测试
2. 边界条件测试
3. 错误处理测试
4. 用户交互测试

## 运行测试

### 运行所有功能测试

```bash
swift test --filter FeaturesTests
```

### 运行特定功能测试

```bash
swift test --filter FeaturesTests.SlideEditingTests
```

## 编写测试指南

1. 每个功能模块创建独立的测试类
2. 使用描述性的测试方法名称
3. 包含必要的测试数据和资源
4. 测试各种用户场景和边界情况

## 维护和更新

- 定期审查并更新测试用例
- 确保测试与最新功能保持同步
- 删除过时的测试用例
- 优化测试性能
