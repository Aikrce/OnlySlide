# 性能测试

## 概述

这个目录包含 OnlySlide 的性能测试用例，用于衡量和监控应用的性能指标。

## 测试范围

- 启动时间测试：衡量应用启动所需的时间
- 渲染性能测试：测量幻灯片渲染的性能
- 内存使用测试：监控内存占用和潜在泄漏
- 数据库性能测试：评估数据存储和检索效率
- 网络性能测试：测量网络请求的响应时间

## 运行测试

执行以下命令运行所有性能测试：

```bash
swift test --filter PerformanceTests
```

要运行特定性能测试：

```bash
swift test --filter PerformanceTests.RenderingPerformanceTests
```

## 性能基准

性能测试会与预设的基准值进行比较，以便及时发现性能退化。基准值存储在 `PerformanceBaselines.json` 文件中。

## 添加新测试

添加新的性能测试时，请遵循以下规则：

1. 使用 `measure` 方法收集性能数据
2. 设置合理的性能基准值
3. 确保测试环境一致，以避免外部因素影响测试结果
4. 添加详细的测试描述，说明测试的目标和指标

## 性能测试报告

性能测试结果会生成报告，存储在 `performance-reports` 目录中，可用于追踪性能变化趋势。
